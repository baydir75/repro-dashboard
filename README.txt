Objectif : Reproduire le dashboard dont le screen est fourni dans le dossier.

Consignes générales :
    - N'essayez pas de reproduire le graphique barré, ça n'est pas l'important (laissez la case vide)
    - Les icônes ne sont pas importantes non plus, utilisez fontawesome et choisissez n'importe lesquelles
    - Faites un design responsive
    - Créez un repo GitHub public spécialement pour cet exercice, et faites des commits après chaque fonctionnalité codée
    - Faites un code propre et bien indenté, avec les bonnes balises HTML, des noms de classes cohérents, et des commentaires

Consignes relatives au CSS :
    - Vous devez utiliser les propriétés CSS suivantes : position, transition, box-shadow, border
    - L'unité px est interdite. Utilisez seulement le rem

Appliquez-vous, et surtout, amusez-vous !
